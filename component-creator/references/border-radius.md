# Border Radius

| Token | Value | Description |
|---|---|---|
| 小 | 2px | Small radius |
| 中 | 4px | Medium radius |
| 标准 | 6px | Standard radius |
| 大 | 8px | Large radius |

| Component | Token | Value |
