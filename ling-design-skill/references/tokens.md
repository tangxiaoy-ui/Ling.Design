# Design Tokens

Ling Design uses a set of global variables for consistency.

## Available Tokens

- **Colors**: Primary brand colors, status colors, and neutral grays. See [colors.md](./colors.md).
- **Spacing**: Margins, paddings, and gaps. See [spacing.md](./spacing.md).
- **Typography**: Font sizes and weights for different text levels. See [typography.md](./typography.md).
- **Border Radius**: Corner radius values for components. See [border-radius.md](./border-radius.md).
